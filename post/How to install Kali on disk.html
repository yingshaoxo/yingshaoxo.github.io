How to install Kali on disk

Kali installation, It's all about choice.

  1. format your disk using PE system, we need a clear disk, or something will going wrong

  2. use refus to make USB driver

  3. just to be safe, use MBR based mix mode

  4. use dd mode to defend Wi-Fi firmware missing (iso file belongs to Windows, and Windows is bad on drivers)

  5. if you can disable secure boot, you can use UEFI in installation, else just use BIOS

  6. if you failed, try more option in -1 and 3 step.
