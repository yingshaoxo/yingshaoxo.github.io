10、万用表量程的选择，数字万用表二极管档的使用。

量程的选择: 被测物的最大值应是满刻度的三分之二。如测量20V的电压，应选用30V的量程。

测二极管: 把量程转换到欧姆档R*100或R*1K。黑表笔接二极管正极，红表笔接二极管负极，正向电阻越小越好。红表笔接正极，黑表笔接负极，电阻无穷大为好。

11、鸟牌功率计的结构、使用方法与场合。

结构: 电源、主机、传感器

使用场合: 测量正向和逆向平均功率、峰值，或突发功率、电压驻波比(VSWR)、振幅因数，及互补累计分布函数(CCDF)等，用来对发射机及天馈系统的性能进行检测和故障判断

使用方法:

  1. 开机
  2. 设定标度
  3. 设定元件型号
  4. 开启射频源，读数
  5. 长按OK键2秒，睡眠。长按OK键8秒，完全关闭设备

12、天馈线分析仪使用场合、端接部件、测试结果分析与故障判断。

使用场合: 快速、准确地提供天馈线分析，如插入损耗/电缆损耗、回波损耗/驻波比、故障定位

端接部件: 开路/短路器、50欧姆端接、天线

故障判断:

  1. 终端接了 50欧姆标准负载: abs(回波损耗) < 30dB
  2. 终端接了 天线: 14dB < abs(回波损耗) < 20dB
  3. 终端接了 开路 或 短路: 0dB < abs(回波损耗) < 5dB

语音组呼: 在任何一个时刻只能有一个成员讲话

语音广播: 只有发起者一个人能讲话

16、CIR设备GSM-R话音单元、数据单元、主控单元的功能。

话音单元:

  1. 在主控单元的控制下自动完成GSM-R网络注册
  2. 在主控单元的控制下自动完成GSM-R话音调度通信
  3. 在主控单元的控制下自动完成GSM-R机车功能号的注册注销
  4. 在主控单元的控制下自动完成GSM-R车次功能号的注册注销

数据单元:

  1. 在主控单元的控制下自动完成GSM-R网络注册
  2. 在主控单元的控制下完成IP地址的获取
  3. 在主控单元的控制下完成GPIS IP地址的申请
  4. 在主控单元的控制下通过GPRS网络采用UDP/IP方式，发送无线车次号校核信息，接收无线调度命令，并发送回执信息

主控单元：

  1. 实现对MMI、记录单元、GPS单元、GSM-R话音单元、GSM-R数据单元、450MHz单元、LBJ单元、接口单元的通信与控制
  2. 接收转发处理各单元的数据，并转发至记录单元
  3. 完成了对多包调度命令信息的拼包处理并让MMI显示
  4. 在 GSM-R 工作模式下，根据车次号，控制无线车次号校核信息的发送
  5. 实现对各路音频的汇接
  6. 完成GSM-R调度通信的语音业务
  7. 完成其他数据业务

17、多频段合路器的功能。

多路复用，让几部发射机或接收机同时工作而又不互相影响

18、450M无线列调系统频率配置。

4频组配置。

同频通话时，机车台和车站台收发频率都定为F4(低频)；

异频通话时，机车接收频率为F2，发射频率为F4(高频频率)，站台相反。

F1、F3也是高端频率，主要在“大三角”通信时使用。

19、机车数据采集编码器功能。

不断地捕获在TAX箱总线上广播的机车运行数据。

FFSK模式下，在捕获到进站、出站、通过、容许等位置信息时，将车次号、机车号、机车速度、机车位置、车重等信息进行编码，以FFSK方法传送给CIR

29、故障处理总则与“五清”。

总规则: 通信段应制定障碍处理程序，当铁路专用无线通信设备发生故障时，维护人员应该及时准确的作出判断，迅速组织修复，努力减少故障时间。对影响行车安全的设备必须采取倒、代等应急措施，减少影响程度。

五清: 清楚 时间、地点、原因、影响范围、处理过程

32、本机IP的获取，当前GRIS IP的获取

本机IP的获取: GPRS数据模块完成网络注册之后，根据APN和机车号向网络申请本机IP地址

GPIS IP的获取:

  1. 本机IP地址获取后，等待GPS信号，一旦有信号，CIR向北京主用GPOS请求当前GRIS IP地址
  2. 若重复三次请求都没有回应，向武汉备用GPOS请求当前GRIS IP地址
  3. 若重复三次请求都没有回应，向自己路局的归属GRIS服务器请求当前GRIS IP地址
  4. 若还是没有回应，则将自己路局的归属GRIS服务器当作当前GRIS 服务器
